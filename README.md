# Yelp

Yelp is a mobile-first disaster reporting MVP prototype.

The current version is a static UI flow demo for Vercel. It shows:

- one-tap Yelp activation
- background status dashboard
- location and signal progress states
- disaster JSON preview
- placeholder modules for gesture input, block-based reporting, and AI interpretation summaries

## Run Locally

Serve the folder with any static server.

Example:

```bash
npx serve . -l 4173
```

## Deploy To Vercel

Import this GitHub repository into Vercel as a static site. No build command is required.
